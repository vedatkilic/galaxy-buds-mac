Galaxy Buds for Mac

Control your Samsung Galaxy Buds from the macOS menu bar.

Download View on GitHub

Features

  • Per-earbud and case battery, right in the menu bar
  • Noise control: Off / Ambient / Adaptive / ANC, plus ANC strength
  • Equalizer and ambient sound customization (per-side level + tone)
  • Touch-and-hold actions, touchpad lock
  • Find My Earbuds and an earbud fit test
  • Auto-connect, launch at login, and 11 languages

Native SwiftUI, no dependencies. Supports Galaxy Buds through Buds4 Pro.

Install

Homebrew

brew tap vedatkilic/tap
brew trust vedatkilic/tap
brew install --cask vedatkilic/tap/galaxy-buds

Or download the .dmg from Releases and drag the app into Applications.

The app is open-source and unsigned, so on first launch right-click it → Open, or allow it under System Settings → Privacy & Security → Open Anyway.

Privacy

No analytics, no tracking, no telemetry, no network calls. The app talks only to your earbuds over Bluetooth, and everything stays on your Mac. The only permission requested is Bluetooth.
